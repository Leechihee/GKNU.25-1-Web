Python


누구나 쉽게 시작할 수 있는 프로그래밍 언어입니다.
문법이 간단하고 읽기 쉬워 코딩이 처음인 분께도 적합합니다.
웹, 데이터, 인공지능까지 모두 파이썬으로 가능합니다.

print


    print("Hello, Python!")

출력 화면


    Hello, Python!

print() 함수는 파이썬에서 화면에 원하는 내용을 출력할 때 사용하는 가장 기본적인 함수입니다.
괄호 안에 출력하고 싶은 내용을 따옴표로 묶어 넣으면, 그 내용이 그대로 화면에 표시됩니다.
예를 들어 print("Hello, Python!")처럼 작성하면, 화면에는 Hello, Python!이라는 문장이 출력됩니다.
프로그래밍을 처음 배울 때 가장 먼저 접하게 되는 함수로, 코드 실행 결과를 확인하는 데 유용하게 사용됩니다.

Variable


    name = "Alice"
    age = 20
    print("이름:", name)
    print("나이:", age)

출력 화면


    이름: Alice
    나이: 20

파이썬에서 변수는 데이터를 저장하는 이름표처럼 사용됩니다.
예를 들어 name = "Alice"처럼 작성하면, name이라는 변수에 "Alice"라는 문자열이 저장됩니다.
이때 파이썬은 변수의 자료형을 따로 지정하지 않아도 자동으로 판단해줍니다.

반면, C++이나 Java에서는 변수를 선언할 때 자료형을 반드시 명시해야 합니다.
예를 들어 C++에서는 string name = "Alice";처럼,
Java에서는 String name = "Alice";처럼 작성해야 합니다.

파이썬은 이러한 자료형 선언 없이도 변수 사용이 가능하기 때문에,
초보자가 빠르게 코드를 작성해보고 실습하기에 매우 편리한 언어입니다.
또한 변수에 저장된 값은 print() 함수를 사용해 쉽게 출력할 수 있으며,
변수의 값은 언제든지 바꿀 수 있습니다.

input


    name = input("이름을 입력하세요: ")
    print("안녕하세요,", name)

출력 예시


    이름을 입력하세요: Alice
    안녕하세요, Alice

input() 함수는 사용자로부터 데이터를 입력받을 때 사용합니다.
위 예제에서는 input("이름을 입력하세요: ")를 통해 사용자에게 이름을 입력받고,
입력된 값을 변수 name에 저장한 뒤 print()로 출력하고 있습니다.

파이썬에서는 입력받은 값이 자동으로 문자열(str)로 처리되며,
숫자로 계산하고 싶을 경우 int()나 float()로 형 변환을 해야 합니다.

C++이나 Java에서는 입력을 받을 때 별도의 입출력 객체를 사용합니다.
예를 들어, C++에서는 cin >> name;, Java에서는 Scanner sc = new Scanner(System.in);
그리고 String name = sc.nextLine();과 같이 입력을 받아야 합니다.

파이썬의 input()은 간단하고 직관적이어서, 입문자가 쉽게 따라할 수 있는 장점이 있습니다.

배열 / Array


    fruits = ["사과", "바나나", "오렌지"]
    print("첫 번째 과일:", fruits[0])
    print("전체 과일 목록:")
    for fruit in fruits:
        print(fruit)

출력 화면


    첫 번째 과일: 사과
    전체 과일 목록:
    사과
    바나나
    오렌지

파이썬에서는 배열과 비슷한 구조로 리스트(list)를 사용합니다.
대괄호 [] 안에 여러 값을 넣어 하나의 변수로 관리할 수 있습니다.
위 예제에서는 fruits라는 리스트에 3개의 과일 이름을 저장하고 있습니다.

fruits[0]처럼 인덱스를 사용하면 특정 위치의 값을 가져올 수 있고,
for 반복문을 사용해 리스트 안의 모든 값을 하나씩 출력할 수도 있습니다.

C++에서는 string fruits[3] = {"사과", "바나나", "오렌지"};처럼 배열을 사용하고,
Java에서는 String[] fruits = {"사과", "바나나", "오렌지"};처럼 선언합니다.
하지만 파이썬의 리스트는 크기를 자유롭게 바꾸거나 다양한 자료형을 함께 넣을 수 있다는 점에서 더 유연합니다.

문자열 / String


    greeting = "Hello"
    name = "Python"
    message = greeting + ", " + name + "!"
    print(message)

출력 화면


    Hello, Python!

파이썬에서 문자열은 "큰따옴표" 또는 '작은따옴표'로 감싸서 표현합니다.
문자열끼리는 + 연산자를 사용해 쉽게 이어붙일 수 있습니다. 이를 문자열 연결(String Concatenation)이라고 합니다.
위 예제에서는 greeting, name이라는 두 문자열을 이어붙여 Hello, Python!이라는 문장을 만들고 출력하고 있습니다.

C++에서는 문자열 연결을 위해 string message = greeting + ", " + name + "!";처럼 사용하며,
Java도 String message = greeting + ", " + name + "!";처럼 동일한 방식으로 문자열을 연결할 수 있습니다.

하지만 파이썬은 자료형 선언 없이도 문자열을 만들 수 있고,
다양한 문자열 함수들(len(), upper(), replace() 등)을 활용해 쉽게 다룰 수 있다는 장점이 있습니다.
