#include


    #include <iostream>
              
    int main() {
        std::cout << "Hello, World!" << std::endl;
        return 0;
    }

출력 화면


    Hello, World!

#include <iostream>는 C++에서 기본 입출력(입력: cin, 출력: cout)을 사용할 수 있게 해주는 명령어입니다.
이 예제는 C++ 프로그램의 기본 구조를 보여주는 "Hello, World!" 출력 코드입니다.
main() 함수는 프로그램의 시작점이고, std::cout을 사용해 화면에 텍스트를 출력합니다.

cout / endl


    #include <iostream>

    int main(){
        std::cout << "콘솔 창에 원하는 내용을 출력하는 함수입니다." << std::endl;
        return 0;
    }

출력 화면


    콘솔 창에 원하는 내용을 출력하는 함수입니다.

cout은 "Character Output"의 줄임말로, 콘솔 창(터미널)에 내용을 보여주는 역할을 합니다.
<< 연산자는 "보낸다"라는 뜻으로, 오른쪽에 있는 값을 왼쪽(cout)에 전달해 출력하는 방식입니다.
예를 들어 위 코드는 사용자가 콘솔 창에서 메시지를 볼 수 있도록 하는 아주 기본적인 출력 방법입니다.
endl은 "End Line"의 줄임말로, 새로운 줄을 만들어주는 역할을 합니다.
같은 의미로 '\n'도 있습니다.

Variable

 
    #include <iostream>
  
    int main(){
        int i = 10;
        double pie = 3.14;
        char ch = 'a';
        bool flag = true;
    
        std::cout << i << std::endl; // 정수형 변수(Integer)
        std::cout << pie << std::endl; // 실수형 변수
        std::cout << ch << std::endl; // 문자형 변수(character)
        std::cout << flag << std::endl; // 진위형 변수(True/False)
    
        return 0;
    }

출력 화면


    10
    3.14
    a
    1 // false은 0, true는 1

변수(Variable)는 데이터를 저장하는 이름 있는 공간입니다.
- int: 정수를 저장할 수 있습니다. 예: 10
- double: 소수점이 있는 숫자, 즉 실수를 저장합니다. 예: 3.14
- char: 한 글자를 저장합니다. 예: 'a'
- bool: 참 또는 거짓(true 또는 false)을 저장합니다.
이 코드에서는 각각의 변수 값을 cout으로 출력하고 있습니다.

cin


    #include <iostream>
  
    int main(){
        int number;
        std::cout << "숫자를 입력하세요. : ";
        std::cin >> number;
        std::cout << "\n입력하신 숫자는 " << number << "입니다.\n";
        return 0;
    }

출력 화면


    숫자를 입력하세요. : 64
    입력하신 숫자는 64입니다.

cin은 "Character Input"의 줄임말로, 사용자로부터 값을 입력받을 때 사용합니다.
>> 연산자는 오른쪽에 있는 변수로 값을 받아오는 동작을 의미합니다.
위 코드는 사용자에게 숫자를 입력하라고 메시지를 출력한 뒤, 입력받은 숫자를 다시 출력해주는 예제입니다.
입력 → 저장 → 출력 흐름을 연습하기 좋은 코드입니다.

pointer


    #include <iostream>
  
    int main(){
        int number = 10;
        int *address;
        address = &number;
        std::cout << "address에 저장된 숫자는" << *address << "입니다\n";
        return 0;
    }

출력 화면


    10

포인터는 집 주소에 비유하면 이해하기가 더 쉽습니다.

예를 들어 어떤 사람의 집에는 책(데이터)이 들어 있습니다. 이 집의 위치를 알려면 '주소'가 필요하겠지요?
컴퓨터도 이와 마찬가지로, 변수에 저장된 값을 찾기 위해 메모리 주소를 사용합니다.

- int number = 10;은 "10이라는 책이 들어 있는 집"이라고 생각할 수 있습니다.
- &number는 그 집의 주소를 의미합니다. 즉, number라는 변수가 메모리 상 어디에 있는지를 나타냅니다.
- int* address;는 그 주소를 저장하는 주소 메모장과 같은 역할을 합니다.
- *address는 메모장에 적힌 주소를 따라가서, 해당 집 안에 있는 책(값)을 가져오는 동작을 의미합니다.

정리하면, 포인터는 "주소를 저장하고 그 주소를 따라가 값을 가져오는 방법"이라고 이해할 수 있습니다.

array


    #include <iostream>
  
    int main(){
        int numArray[3];
        numArray[0] = 1;
        numArray[1] = 10;
        numArray[2] = 100;
        // num Array[3] = 100; 배열의 크기가 3개이기 때문에 더 넣으면 Index Error
        std::cout << numArray[0] << ' ' << numArray[1] << ' ' << numArray[2] << '\n';
        return 0;
    }

출력 화면


    1 10 100

배열(array)은 같은 종류의 데이터를 한 줄로 정리해 놓은 것이라고 생각하면 이해하기 쉽습니다.

예를 들어, 학생들의 시험 점수를 저장한다고 할 때, int score1, score2, score3;처럼 하나하나 변수를 만들 수도 있지만, int scores[3];처럼 배열을 사용하면 훨씬 더 간단하게 처리할 수 있습니다.

위 코드에서는 int numArray[3];를 통해 정수형 값을 3개 저장할 수 있는 배열을 만들었습니다.

- numArray[0] = 1; : 배열의 첫 번째 칸에 1을 저장합니다 (C++에서 배열은 0부터 시작해요).
- numArray[1] = 10; : 두 번째 칸에 10을 저장합니다.
- numArray[2] = 100; : 세 번째 칸에 100을 저장합니다.

만약 numArray[3]처럼 없는 칸에 값을 넣으려고 하면, 프로그램이 잘못된 메모리에 접근하게 되어 오류(Index Error)가 발생합니다.

정리하자면, 배열은 동일한 타입의 데이터를 연속적으로 저장할 수 있는 공간이며, 인덱스를 통해 하나하나 접근할 수 있습니다. 주의할 점은 인덱스는 0부터 시작하고, 배열의 크기를 넘지 않아야 한다는 것입니다!

문자열 / string


    #include <iostream>
    #include <string>
  
    int main(){
        char cstr[100] = "C스타일 문자열"; // C에서는 char 배열로 문자열을 표현
        std::string str = "C++스타일 문자열"; // C++에서는 string이 정의되어 있기 때문에 string을 사용

        std::cout << "C 스타일 문자열 : " << cstr << std::endl;
        std::cout << "C++ 스타일 문자열 : " << str << std::endl;
        return 0;
    }

출력 화면


    C 스타일 문자열 : C스타일 문자열  
    C++ 스타일 문자열 : C++스타일 문자열

문자열은 글자들을 모아놓은 데이터라고 생각하시면 됩니다. 예를 들어 "안녕하세요", "Hello" 같은 말들이 전부 문자열입니다.

C++에서는 문자열을 표현하는 방법이 두 가지가 있습니다.

C 스타일 문자열 - char cstr[100] = "C스타일 문자열";
- 이건 문자 하나하나를 배열로 저장하는 방식입니다.
- 즉, 글자들을 char라는 자료형의 배열에 넣는 거예요.
- 끝에 '\0'라는 특수문자가 자동으로 붙어서 문자열의 끝을 표시합니다.

C++ 스타일 문자열 - std::string str = "C++스타일 문자열";
- C++에서는 string이라는 더 편한 문자열 자료형이 따로 제공됩니다.
- 문자열끼리 더하거나, 길이를 재거나, 비교하는 기능도 훨씬 간단하게 사용할 수 있어요.

new/delete


    #include <iostream>
  
    int main(){
        int newExampleAddress = new int(100);
        std::cout << *newExampleAddress << '\n';
        delete newExampleAddress;
        return 0;
    }

출력 화면


    100

프로그램 실행 중에 새로운 공간을 만들어서 데이터를 저장하고 싶을 때 사용하는 new와 delete에 대해 알아보겠습니다.

쉽게 말해, new는 집을 새로 짓는 것이고, delete는 그 집을 철거하는 것이라고 볼 수 있습니다.

- new int(100);은 100이라는 값을 담을 수 있는 집을 새로 짓고, 그 주소를 반환합니다.
- int* newExampleAddress는 그 새로 지은 집의 주소를 받아 저장합니다.
- *newExampleAddress는 그 집 안에 들어 있는 값, 즉 100을 꺼내서 출력합니다.
- delete newExampleAddress;는 다 쓴 집(메모리 공간)을 철거합니다. 메모리는 한정되어 있기 때문에 다 쓴 공간은 꼭 이렇게 정리해 줘야 합니다

정리하자면, new는 동적으로 메모리를 생성할 때 사용하고, delete는 그 메모리를 정리할 때 사용합니다.

만약 delete를 하지 않으면 컴퓨터가 기억해야 할 정보가 계속 쌓여서, 메모리 누수라는 문제가 생길 수도 있습니다.
