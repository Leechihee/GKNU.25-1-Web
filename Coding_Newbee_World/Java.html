Java


Java는 전 세계에서 널리 사용되는 쉽고 강력한 프로그래밍 언어입니다.
한 번 작성하면 어디서든 실행 가능한 유연함을 갖췄습니다.
앱, 웹, 게임, 서버 개발까지 — Java로 무엇이든 시작해봅시다.

System.out.println


    public class Main {
        public static void main(String[] args) {
            System.out.println("콘솔 창에 원하는 내용을 출력하는 함수입니다.");
        }
    }

출력 화면


    콘솔 창에 원하는 내용을 출력하는 함수입니다.

Java에서는 System.out.println()을 사용해 출력합니다.
println은 출력 후 줄바꿈을 해주며, 줄바꿈 없이 출력하려면 print를 사용합니다.
\n을 사용하면 줄바꿈 효과도 낼 수 있습니다.

Variable


    public class Main {
        public static void main(String[] args) {
            int i = 10;
            double pie = 3.14;
            char ch = 'a';
            boolean flag = true;
    
            System.out.println(i);    // 정수형 변수
            System.out.println(pie);  // 실수형 변수
            System.out.println(ch);   // 문자형 변수
            System.out.println(flag); // 논리형 변수
        }
    }

출력 화면


    10
    3.14
    a
    true

변수(Variable)는 데이터를 저장하는 이름 있는 공간입니다.
- int: 정수를 저장할 수 있습니다. 예: 10
- double: 실수를 저장합니다. 예: 3.14
- char: 한 글자를 저장합니다. 예: 'a'
- boolean: true 또는 false 값을 저장합니다.
Java는 System.out.println()을 이용해 콘솔에 값을 출력합니다.

Scanner


    import java.util.Scanner;
    
    public class Main {
        public static void main(String[] args) {
            Scanner scanner = new Scanner(System.in);  // 사용자 입력을 받기 위한 Scanner 객체 생성
            System.out.print("숫자을 입력하세요: ");
            int number = scanner.nextInt();          // 사용자로부터 문자열 입력 받기
            System.out.println("입력하신 숫자는" + number + "입니다.");
        }
    }

출력 예시


    숫자을 입력하세요: 128
    입력하신 숫자는 128입니다.

Java에서는 프로그램에서 특정 기능(예: 입력, 날짜, 리스트 등)을 사용하려면 관련된 클래스를 불러와야 합니다.
이때 사용하는 것이 바로 import 문입니다.

Scanner는 키보드로부터 입력을 받기 위한 도구인데, 이 도구는 java.util이라는 곳에 들어 있어
import java.util.Scanner;라고 작성해서 불러옵니다.

참고: 자바에서는 기본적인 기능은 바로 사용할 수 있지만, Scanner처럼 외부 클래스는 꼭 import를 해줘야 합니다.

배열 / Array


    public class Main {
        public static void main(String[] args) {
            int[] numArray = new int[3];
            numArray[0] = 1;
            numArray[1] = 10;
            numArray[2] = 100;
        
            System.out.println(numArray[0] + " " + numArray[1] + " " + numArray[2]);
        }
    }

출력 화면


    1 10 100

Java의 배열은 int[] arr = new int[3];처럼 선언합니다.
인덱스는 0부터 시작하며, 범위를 벗어나면 ArrayIndexOutOfBoundsException 오류가 발생합니다.

문자열 / String


    public class Main {
        public static void main(String[] args) {
            char[] cstr = {'J', 'a', 'v', 'a'}; // 문자 배열
            String str = "Java 문자열";         // 문자열
        
            System.out.println("char 배열: " + new String(cstr));
            System.out.println("String 객체: " + str);
        }
    }

출력 화면


    char 배열: 문자 배열
    String 객체: Java 문자열

Java에서는 문자열을 String 클래스로 다룹니다.
char[] 배열을 문자열처럼 쓸 수 있지만, 대부분의 경우 String 클래스를 사용합니다.
new String(char[])을 이용하면 문자 배열을 문자열로 변환할 수 있습니다.

클래스 (class)


    public class Site {
        // 속성 (필드)
        String name;
        int age;
    
        // 행동 (메서드)
        void sayHello() {
            System.out.println("안녕하세요, 사이트 이름은 " + name + "이고, 만들어진지 " + age + "일입니다.");
        }
    }

    public class Main {
        public static void main(String[] args) {
            Site p = new Site();  // Site 클래스의 객체 생성
            p.name = "CNW";
            p.age = 7;
            p.sayHello();             // 메서드 호출
        }
    }

출력 예시


    안녕하세요, 사이트 이름은 CNW이고, 만들어진지 7일입니다

클래스(class)는 현실 세계의 사물이나 개념을 코드로 표현한 설계도입니다.
예를 들어 "사람"이라는 개념을 표현하기 위해 Site이라는 클래스를 만들 수 있어요.

클래스 안에는:

  • 속성(필드): 클래스가 가지고 있는 정보 (예: 이름, 나이)
  • 행동(메서드): 클래스가 할 수 있는 동작 (예: 인사하기, 말하기)
위 코드에서는:
  • String name, int age는 사람의 정보를 담는 변수이고,
  • sayHello()는 인사하는 동작을 수행하는 메서드입니다.
new를 사용해서 클래스로부터 실제 객체(=인스턴스)를 만들 수 있습니다.
이 객체는 설계도를 바탕으로 만들어진 실제 사용 가능한 대상이에요.
